# Output Folder Structure

## Folder Organization

```
output/
├── bin/              Compiled executables
│   └── compile.exe   Build tool (generated by npm run build-exe)
├── builds/           Game ROM files
│   ├── LabyrinthOfTheDragon.gbc   Main game ROM
│   └── *.gbc         Additional game builds
├── dist/             Compiled output
│   └── index.js      Compiled TypeScript/JavaScript
├── obj/              Object files
│   └── *.asm         Assembly listings
├── docs/             Documentation
│   ├── FEATURE_TESTING_GUIDE.md
│   ├── API_INTEGRATION_GUIDE.md
│   └── ...           Other documentation
└── source/           Source code
    ├── ts/           TypeScript source
    ├── c/            C source code
    └── js/           JavaScript source

```

## How to Build

1. **Build Everything:**
   ```
   npm run build-exe
   .compile.exe all
   ```

2. **Build Game Only:**
   ```
   .compile.exe game
   ```

3. **Organize Output:**
   ```
   npm run setup-output
   ```

## Output Files

After building, you'll find:
- **compile.exe** → output/bin/compile.exe
- **LabyrinthOfTheDragon.gbc** → output/builds/LabyrinthOfTheDragon.gbc
- **Compiled JS** → output/dist/
- **Documentation** → output/docs/
